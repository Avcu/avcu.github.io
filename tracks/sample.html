I love walking, running and biking. On this page, I visualized the routes that I have taken. You can also find several links to the tools that I used to prepare this page.


Technical details: Preparing this website is two steps. First, I wrote a small data pipeline in Python that processes given GPX (this is how fitness/tracking apps store your geospatial data) files, and creates a KML file with all the tracks and a JSON file to store aggregate statistics. Then, the KML file is provided to Maps Javascript API, while we load the JSON file to display the statistics.
